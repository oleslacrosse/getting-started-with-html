How the Web Works

If I want to check out my favorite site, I would first have to type the site name in the search bar. My browser will then make a request via the internet using a simple http protocol which allows it to communicate with servers. Servers are computers that host files that make up the web. These files are written in a language called HTML which stands for hypertext markup language. Once the server has recieved the browers' request, the servers will respond with files that the brower can interpret and display to end user. Check out this video for more information.

HTML

HTML documents are coded webpages that contain text content (what you see), references to other documents (images and video), and/or links to other pages. These documents are created using HTML markup tags. Some of the most common tags are written with a start tag, and ending tag, and content in between. Tags that don't contain content are called void tags.

Inline vs. Block Elements

Inline and block elements are terms that explain why there are different ways of making new lines. Inline elements end the line whereever tag is written and wraps any additional text to the next line. On the other hand, block elements create an invisible box around text that falls between the start and end tags. The paragraph tag is a great example of a block element. Check out this video for more information on inline and block elements.
